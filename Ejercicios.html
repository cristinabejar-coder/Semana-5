EJERCICIO 3

Enunciado:Confeccionar una lista de definición indicando los conceptos de distintos lenguajes de programación (por lo menos mencione cuatro lenguajes de programación). En la definición hacer una breve introducción del mismo.

Python
Es un lenguaje de alto nivel, conocido por su sintaxis sencilla y muy fácil de leer. Es extremadamente versátil y se utiliza ampliamente en inteligencia artificial, ciencia de datos, automatización y desarrollo web backend.
JavaScript
Es el lenguaje fundamental de la web. Se ejecuta directamente en el navegador del usuario y permite agregar interactividad, animaciones y comportamiento dinámico a las páginas web. Hoy en día también se utiliza en servidores mediante entornos como Node.js.
Java
Es un lenguaje orientado a objetos diseñado para ser portátil (bajo el lema "escribe una vez, ejecuta en cualquier lugar"). Es muy robusto, seguro y se utiliza comúnmente en el desarrollo de aplicaciones empresariales a gran escala, sistemas bancarios y aplicaciones nativas para Android.
C++
Es una extensión del lenguaje C clásico que añade características más modernas, como la programación orientada a objetos. Destaca por ofrecer un altísimo rendimiento y un gran control sobre el hardware de la computadora, por lo que es indispensable en la creación de videojuegos, motores gráficos y sistemas operativos.
